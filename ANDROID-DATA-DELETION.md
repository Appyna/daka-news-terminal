# SUPPRESSION DES DONNÉES - DAKA NEWS

## Vos droits sur vos données

Conformément au RGPD, vous pouvez demander :

1. **L'accès** à vos données
2. **La modification** de vos données
3. **La suppression** de vos données
4. **L'export** de vos données

---

## Comment supprimer vos données ?

### Option 1 : Suppression totale du compte

Voir : [Guide de suppression de compte](https://github.com/Appyna/daka-news-terminal/blob/master/ANDROID-ACCOUNT-DELETION.md)

**⚠️ Supprime TOUTES vos données définitivement.**

---

### Option 2 : Suppression partielle

Si vous voulez **garder votre compte** mais supprimer certaines données :

**Envoyez un email à :** dakanewsapp@gmail.com

**Objet :** "Suppression de données partielles"

**Précisez ce que vous voulez supprimer :**
- Historique de lecture
- Préférences
- Données d'abonnement
- Autres

**Délai :** 7 jours ouvrés

---

## Qu'est-ce qui est conservé ?

Si vous supprimez seulement certaines données :

- ✅ Votre compte reste actif
- ✅ Vous gardez votre accès Premium (si abonné)
- ✅ Votre email reste dans notre base (pour la connexion)

---

## Export de vos données

Pour recevoir une copie de vos données au format JSON :

**Email :** dakanewsapp@gmail.com  
**Objet :** "Export de mes données"  
**Délai :** 30 jours maximum
